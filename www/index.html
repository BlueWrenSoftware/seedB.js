        Nodejs and Electron Installations

        Software developed using Ubuntu 20.04 and Electron for deployment.

        It is recommended to install nodejs using NVM (node version manager). To install NVM using curl, run the command:

        curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.37.2/install.sh | bash

        More detailed instructions for the installation of NVM (Node Version Manager).

        Once NVM is installed, the latest Nodejs can be installed with:

        nvm install --lts

        Check Nodejs installation with the following commands:

        nvm use --lts
        node -v
        npm -v

        In the terminal open directory ./www/electron and install Electron with the following commands:

        nvm use --lts
        npm install --save-dev electron

        To start Electron use the following commands in the /www/electron directory:

        nvm use --lts To make sure system Nodejs is not used or no system Nodejs is present.

        npm start
